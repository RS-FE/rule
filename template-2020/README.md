# 大屏页面

## 项目结构

``` shell 
─lib                      # 第三方库
├─module                  # 模块静态资源      
│  ├─css                    # 层叠样式表 
│  ├─img                    # 图片【按模块】
│  ├─js                     # 脚本【按模块】
│  └─scss                   # scss【按模块 入口文件编译到 css 目录】
├─page                    # 模块页面
└─public                  # 全局静态资源
```

## 命名规范

文件命名、CSS 类名均采用横线连接 (kebab-case) ，js 中可采用小写驼峰命名法。
